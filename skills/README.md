# Skills

Modular capabilities that extend what agents and executives can do.

## Current Skills

| Skill | Purpose | Used By |
|-------|---------|---------|
| `hire-executive` | Create new executive agents | CEO |
| `hire-agent` | Create new functional agents | Executives |
| `linkedin-post` | Create LinkedIn posts following brand playbook | Content Writer |
| `email-sequence` | Create multi-email sequences | Email Marketing Specialist |

| `content-brief` | Create standardized content briefs | Content Strategist |
| `pitch-deck` | Generate pitch deck content | CMO |
| `case-study` | Create case studies from results | CMO, Content Writer |

## Adding Skills

Create `/skills/[skill-name]/SKILL.md` with:
- Purpose and when to use
- Prerequisites
- Step-by-step process
- Templates and examples
- Self-review checklist
- Required context files

## Key Principle

All content-producing skills load brand-guidelines.md, tone-of-voice.md, and messaging-framework.md as their foundation. Channel-specific adaptations come from the relevant playbook file. This ensures cohesive brand voice with appropriate platform variations.
