# Skills

Modular capabilities that extend what agents and executives can do.

## Current Skills

| Skill | Purpose | Used By |
|-------|---------|---------|
| `hire-executive` | Create new executive agents | CEO |
| `hire-agent` | Create new functional agents | Executives |

## Adding Skills

Create `/skills/[skill-name]/SKILL.md` with:
- Purpose and when to use
- Prerequisites
- Step-by-step process
- Examples
