# Blog Post Skill

Create blog posts following brand voice, content pillars, SEO strategy, and blog playbook.

## When to Use

When Content Writer needs to create blog content based on a brief.

## Prerequisites

- Brand guidelines exist (`/context/marketing/brand-guidelines.md`)
- Tone of voice defined (`/context/marketing/tone-of-voice.md`)
- Messaging framework exists (`/context/marketing/messaging-framework.md`)
- Blog playbook exists (`/context/marketing/playbooks/blog-playbook.md`)
- SEO keyword strategy exists (`/context/marketing/seo-keyword-strategy.md`)

## Process

1. Load context: brand-guidelines.md, tone-of-voice.md, messaging-framework.md, blog-playbook.md, seo-keyword-strategy.md
2. Identify content pillar, target keyword, and funnel stage from brief
3. Select post type from playbook (see templates below)
4. Create outline following playbook structure
5. Draft post with SEO considerations (keyword in title, H2s, first paragraph)
6. Apply tone modifiers for blog channel
7. Include internal CTAs per playbook guidance
8. Self-review against checklist
9. Save to `/outputs/content/blog/`

## Post Type Templates

### How-To / Guide
- Headline: "How to [achieve outcome] [without common pain point]"
- Structure: Problem intro → Step-by-step process → Common mistakes → Key takeaway → CTA
- Length: 1000-1500 words

### Thought Leadership
- Headline: "[Contrarian take] about [industry topic]"
- Structure: Bold opening claim → Evidence and reasoning → Implications → What to do about it → CTA
- Length: 800-1200 words

### Industry Analysis
- Headline: "The State of [topic]: [key finding]"
- Structure: Key data point → Context → Multiple perspectives → Our take → What it means for readers → CTA
- Length: 1200-1800 words

### Case Study / Customer Story
- Headline: "How [customer type] [achieved result] with [approach]"
- Structure: Customer context → Challenge → Solution → Results (quantified) → Lessons → CTA
- Length: 800-1200 words

### Comparison / Versus
- Headline: "[Option A] vs. [Option B]: [what you need to know]"
- Structure: Context for the comparison → Criteria → Side-by-side analysis → Recommendation → CTA
- Length: 1000-1500 words

## Self-Review Checklist

- [ ] Target keyword in title, first paragraph, and at least one H2
- [ ] Compelling headline with clear benefit
- [ ] Hook opening (first 2 sentences grab attention)
- [ ] Scannable structure (H2s, H3s, bullet points)
- [ ] Aligns with content pillar
- [ ] Brand voice consistent
- [ ] Messaging aligns with framework
- [ ] Clear CTA at end
- [ ] Internal links where relevant
- [ ] 800-1800 words (per type)
- [ ] No jargon without context
